--- layout: piecrust_doc title: Documentation --- Using **PieCrust** is mostly a matter of editing text files. Configurations are written in [YAML][], a simple human-readable syntax. Templating is handled by default by [Twig][], with a syntax similar to [Jinja][] or [Django][]. Text formatting is done by default through [Markdown], which lets you write page contents like you would naturally write an email. [yaml]: http://en.wikipedia.org/wiki/YAML [twig]: http://www.twig-project.org/ [jinja]: http://jinja.pocoo.org/ [django]: http://docs.djangoproject.com/en/dev/topics/templates/ [markdown]: http://daringfireball.net/projects/markdown/
