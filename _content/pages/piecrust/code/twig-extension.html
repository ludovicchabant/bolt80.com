--- title: Extending Twig layout: piecrust_code --- To add new features to **PieCrust**'s default template engine, Twig, you can override the `getTwigExtensions` method in your plugin and return an instance of a Twig extension. To learn how to implement Twig extensions, [read the documentation on the official Twig website][1]. [1]: http://twig.sensiolabs.org/doc/extensions.html
